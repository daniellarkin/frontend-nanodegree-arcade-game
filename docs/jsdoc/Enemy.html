update(dt)

Update the enemy object's position. The enemy object should wrap around to the left hand side. When wraping from the right hand side to the left hand side, rather than set the x coord to 0, setting it to a negative value looks more visual pleasing. This has the visual effect of the enemy gradually moving onto the screen gradual from the left hand side rather than abructly reappearing.
Parameters:
Name Type Description
dt number a game engine time delta between ticks. The dt parameter which will ensure the game runs at the same speed for all computers.
